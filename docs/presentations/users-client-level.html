06 Edit access in one action

Add or remove several groups at once.

Click the pencil on any user to open their data groups as a single checklist. Tick what they need, untick what they don't, Save once — no more editing the same person group by group.

  • Search within the client's groups.
  • Every change to a user's access happens in one place.

08 Why it helps you

Less hunting, fewer mistakes, faster onboarding.

Everyone in one place

All 360 of a client's users in a single grid — no more opening data groups one by one.

Full access at a glance

Every data group a person belongs to sits on their row (N : M) — nothing hidden.

Edit access in one action

Answer access questions fast

One source of truth

09 Before → after

The same job, a much shorter path.

Before · per data group
  • Users lived inside each data-group tab
  • A user in N groups = managed in N places
  • No single view of a user's full access
  • Repetitive, error-prone, hard to audit
After · client level
  • One Users grid on the client
  • Each user once, with all groups (N : M)
  • Search, filter by group, paginate hundreds
  • Edit membership & bulk-create with permissions
